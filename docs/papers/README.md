# 研究论文

本目录收录GUI Agent领域的重要研究论文及其解读。

## 内容规划

- **重要论文解读** - 里程碑式论文的详细解读
- **方法论分析** - 不同研究方法的对比分析
- **实验结果对比** - 各方法在基准测试上的表现
- **研究趋势总结** - 领域研究方向和热点分析

## 论文分类

建议按以下维度组织论文：
- 按年份：2020, 2021, 2022, 2023, 2024, 2025
- 按会议：NeurIPS, ICML, CVPR, ACL, CHI等
- 按主题：视觉理解、动作规划、端到端学习等

## 贡献指南

欢迎分享论文解读。建议包含：
- 完整引用信息（标题、作者、会议/期刊、年份）
- 论文链接（arXiv、官网、GitHub等）
- 核心创新点
- 方法概述
- 实验结果
- 个人评价与思考
