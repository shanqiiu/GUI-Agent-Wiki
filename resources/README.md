# 资源文件

本目录存放Wiki相关的资源文件。

## 目录说明

### images/
存放文档中使用的图片资源：
- 架构图
- 流程图
- 实验结果图表
- 工具截图
- 论文插图等

### datasets/
存放数据集相关信息：
- 公开数据集链接
- 数据集说明文档
- 数据格式规范
- 标注指南等

### benchmarks/
存放基准测试相关内容：
- 基准测试结果
- 性能对比表格
- 测试方法说明
- 评估指标定义等

## 使用规范

1. **图片命名**：使用描述性名称，如 `architecture-overview.png`
2. **文件大小**：单个图片不超过5MB，建议压缩后上传
3. **版权说明**：引用外部资源需注明来源
4. **目录组织**：相关文件可创建子目录分类存放
